# PandABlocks-FPGA

PandABlocks-FPGA contains the firmware that runs on the FPGA inside the Zynq
module at the heart of a PandABlocks-enabled device such as PandABox.

The firmware is assembled from a set of functional **blocks** (PULSE, CLOCK,
PCAP, SEQ, COUNTER, LUT, …), each described by an `.ini` and built into an app
for a particular hardware target.
