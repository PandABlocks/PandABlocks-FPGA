# How-to Guides

Practical step-by-step guides for the more experienced user.

To get new functionality onto a PandA, the typical pipeline is:
write a block ([](/how-to/block.md)) → add it to an app ([](/how-to/app.md)) →
build the FPGA image ([](/how-to/build-fpga-image.md)) → install and select the
bitstream on the device
([meta-panda docs](xref:meta-panda/how-to/choose-fpga-bitstream)).
