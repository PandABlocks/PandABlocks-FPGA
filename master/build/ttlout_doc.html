TTLOUT - TTL Output¶

The TTLOUT block handles the signals to the TTL Output connectors

Fields¶

Name Type Description
QUARTER_DELAY param uint 3 Number of 1/4 ticks to delay (range 0-3)
FINE_DELAY param uint 31 Fine delay (range 0-31)
VAL bit_mux TTL output value
